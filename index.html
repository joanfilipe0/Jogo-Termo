Jogo Termo

Regras do Jogo:

1. Escolha o número de letras para a palavra, variando de 3 a 7. Clique no botão "Iniciar Jogo" para começar.

2. Após iniciar o jogo, você verá as caixas para inserir suas tentativas. Cada linha representa uma tentativa, e você tem um número limitado de tentativas.

3. Depois de preencher as letras em uma linha, clique no botão "Verificar" para verificar sua tentativa.

4. As caixas com letras corretas ficarão com fundo verde, letras erradas ficarão com fundo vermelho, caixas amarelas indicam letras corretas, mas na posição errada.

5. Se você adivinhar a palavra antes de atingir o número máximo de tentativas, você ganha.
